# Bank Loan Management System

A relational database platform designed to manage borrowers, loans, repayments, collateral tracking, and repayment analytics.

## Features

- Normalized relational database schema
- Loan and borrower management
- Repayment tracking
- Collateral management
- Audit logging using database triggers
- Stored procedures and user-defined functions
- Risk monitoring using SQL analytics
- Overdue installment detection

## Technologies

- SQL Server 2022
- T-SQL
- Stored Procedures
- User Defined Functions (UDFs)
- Triggers
- Views

## Database Components

### Core Tables

- Borrowers
- Loans
- Loan Officers
- Branches
- Repayments
- Repayment Schedule
- Collateral

### Analytics Views

- Active Loans by Borrower
- Repayment Summary
- Overdue Installments

### Advanced Features

- Audit Logging
- EMI Calculation Function
- Repayment Stored Procedure
- Risk Analysis Cursor
